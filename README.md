# Tour Booking API

A backend application built using Node.js and Express.js that provides a complete tour booking system with authentication, payments, and secure API design.

## 🚀 Features
- User authentication and authorization (JWT)
- Tour booking system with Stripe integration
- RESTful APIs with filtering, sorting, and pagination
- Server-side rendering using Pug
- Email notifications using Nodemailer
- Security features (Helmet, rate limiting, data sanitization)
- Global error handling

## 🛠️ Tech Stack
- Node.js
- Express.js
- MongoDB (Mongoose)
- Stripe API
- Pug
- Nodemailer

- Designing scalable REST APIs
- Implementing authentication & authorization
- Handling payments securely
